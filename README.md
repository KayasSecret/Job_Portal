## ✨ About

JobHub is a modern MERN Stack Job Portal where students can search and apply for jobs while recruiters can post jobs and manage applicants.

Built using React, Node.js, Express.js, MongoDB, Redux Toolkit and Cloudinary.

## 🎯 Features

✅ Secure Authentication

✅ Job Search & Filters

✅ One Click Apply

✅ Resume Upload

✅ Recruiter Dashboard

✅ Company Management

✅ Applicant Tracking

✅ Responsive Design

✅ Modern UI Animations

## 🛠 Tech Stack

⚛ React.js

⚡ Vite

🎨 Tailwind CSS

🔄 Redux Toolkit

🟢 Node.js

🚀 Express.js

🍃 MongoDB

☁ Cloudinary

🔐 JWT Authentication
